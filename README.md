
# Tsume :nail_care:

React app for tracking and organize nail polishes/manicures

:exclamation: This is a Work in Progress :exclamation:



## Installation

### Install dependencies with yarn

```bash
  yarn install
```

### Starting the server

```bash
  yarn start
```

## Run Storybook

```bash
  yarn storybook
```
    
## Running Tests

### Run unit tests

```bash
  yarn test
```

### Run e2e tests

```bash
  yarn playwright
```
Add `--headed` flag for headed mode

### Run Storybook tests (it verifies whether the story renders without any errors)

```bash
  yarn test-storybook
```


## Tech Stack

- React
- JavaScript/TypeScript
- SASS
- React Testing Library
- Playwright
