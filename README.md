
# Tsume

React app for tracking and organize nail polishes/manicures
This is a Work in Progress



## Installation

Install dependencies with yarn

```bash
  yarn install
```

Starting the server

```bash
  yarn start
```
    
## Running Tests

Run unit test

```bash
  yarn test
```

Run e2e test

```bash
  yarn playwright
```
Add `--headed` flag for headed mode


## Tech Stack

- React
- JavaScript/TypeScript
- SASS
- React Testing Library
- Playwright
